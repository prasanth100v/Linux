# ✅ What is Linux?

Linux is a free and open-source operating system (OS) based on the Unix
architecture.\
Created by Linus Torvalds in 1991, it is widely used for servers, cloud
platforms, and development because it is secure, stable, and
customizable.

Some key benefits: - Free to use\
- Strong security features\
- Runs well on low-spec hardware\
- Variety of distributions like Ubuntu, CentOS, and Debian\
- Privacy-friendly\
- Huge community support

Unlike Windows or macOS, Linux is highly customizable, secure, and
efficient.

------------------------------------------------------------------------

## 🔹 How Linux Works

An operating system like Linux works as a bridge between hardware and
software.

-   Hardware cannot understand human instructions directly.\
-   Applications cannot directly control hardware.

Linux takes requests from software, translates them into machine
instructions, and tells the hardware what to do.

**Example:**\
If you play a video, the application asks the OS to use the CPU, memory,
and speakers. Linux manages and coordinates that process.

------------------------------------------------------------------------

## 🔹 Main Components of Linux

1.  **Kernel** -- The core of Linux that interacts directly with
    hardware and manages resources like CPU, memory, and devices.
2.  **Shell** -- The interface that takes commands from the user and
    passes them to the kernel.
3.  **File System** -- Organizes and stores data in a structured way.
4.  **System Utilities & Applications** -- Tools and programs that
    perform tasks like file management, networking, and text editing.

------------------------------------------------------------------------

## 🔹 Real-Life Examples

-   **Servers** -- Most web servers (Google, Facebook, Amazon) run on
    Linux.
-   **Cloud** -- AWS EC2 instances often use Amazon Linux or Ubuntu.
-   **Smartphones** -- Android is built on the Linux kernel.
-   **IoT Devices** -- Smart TVs and routers use Linux.
